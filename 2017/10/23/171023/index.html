공부한 내용

  1. 리눅스 명령어

      1. vi 편집기에서 블록 지정: v
      1. 블록 지정한 후 지우기: d
      1. 파일이름 변경: mv 기존 이름 새 이름

  2. git

      1. 변경 이력 보기: git log -p
      1. 충돌 났을 때는 수동으로 내용 변경해줌.

  3. Junit

      1. 테스트코드 실행시 nullPointerException 발생
        원인: setup()을 junit3 버전으로 만들어서. 아래가 junit4버전.
      1. 테스트 메소드 이름은 자유롭게 쓸 수 있다.
      1. 마우스를 메소드에 대고 메소드 하나씩 테스트할 수 있다.
      1. 테스트에서 객체 내부의 데이터를 비교해야 할 때는 toString()을 오버라이드해서 값을 표기한다.
      1. 같은 데이터를 가진 객체로 테스트 결과를 확인하려면 equals()를 오버라이드.
