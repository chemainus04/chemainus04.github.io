오늘 한 일

  1. slipp 코드리뷰 읽음
    나중에 한 번에 정리해야겠다.

  2. Hue 한테 vi 명령어 배움
    -a: 커서 다음 칸에 쓰기
    -A: 그 줄의 맨 끝에 쓰기
    -o: 아랫줄에 쓰기
    -O: 윗줄에 쓰기
    -I: 맨 앞에 쓰기
    -dd: 지움
    -d2d: 2줄 지움
    -x: delete
    -X: backspace
    -yy: 복사
    -y2y: 2줄 복사
    -p: 붙여넣기
    -b: 해당 단어 앞에 커서
    -e: 해당 단어 끝에 커서
    -2e: 두 단어 끝에 커서
    -y2e: 단어 2개 복사
    -gg: 문서 맨 윗줄로 이동
    -G: 문서 맨 끝줄로 이동
    -H: page up
    -L: page down
    -/단어검색 후 n:다음 단어, N:이전 단어
    -{: 해당 문단 위 }: 해당 문단 아래

  3. 중첩 클래스와 익명 객체 공부

    slipp에 random 값을 사용하면 테스트하기 어려우니까 Random을 상속하는 익명 클래스를 쓰면 좋을 것 같다는 말이 있었다. 거기에 쓰인 코드를 이해하지 못해서 기본서로 공부했는데 시간이 꽤 걸렸다. 아직 더 볼 설명이랑
    연습문제가 남았다. 개념은 얼핏 알겠는데 내가 필요할 때 쓸 수 있을 정도로 이해하진 못 했다.

느낀점

객체지향이 참 어렵다. 자바에 대해 조건문, 반복문 말고 제대로 아는 게 없다.
다시 새로 배우는 기분인데 ‘오오 이런 게 있었구나’ 하면서 재밌기도 하고
너무 배울게 많아서 ‘아 왜 이렇게 많냐’ 싶기도 하다.

내일 할 일

오늘 로또 설계랑 구현을 하나도 안 해서 해야 하는데, 한편으론 자바 기본서 정독하고 연습문제 풀고 싶기도 하다.
둘 다 하기엔 둘 다 시간이 오래 걸리고. 어차피 로또에서도 상속, 컬렉션 써야되서 한 번 쭉 공부는 해야할 것 같은데..
일단 내일은 설명 한 번 읽고 연습문제를 풀어보자.
